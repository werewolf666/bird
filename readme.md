# About Bird

Bird项目是基于laravel 5.4 + Vue 开发的前后端伪分离的webApp

## 主要功能
- 接受用户报名参加赛鸽指定赛，提供报名页面
- 管理员管理比赛信息
- 自动报名参加赛鸽指定赛
## 开发流程
- api 开发
- 后台开发
- 前台开发
